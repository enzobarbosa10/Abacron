O Abacron é uma plataforma de organização financeira pessoal que permite:
- Importação e categorização automática de transações bancárias
- Visualização de dados financeiros através de gráficos e relatórios
- Definição e acompanhamento de metas financeiras
- Exportação de dados em formato CSV
